Category : Graphics / Using React, Context API, styled-components & CSS Grid
Client : Boot Camp
Completion : September 2019
Role : Developer
Github Repository Link : https://github.com/LMTrain/booksearch
Deployed Link : https://lmbooksearch.herokuapp.com/

Description : This is a React-based Books Search app. In this app, React components are created that works with helper/util functions, and utilize React lifecycle methods to query and display books based on user searches. This app also incorporate Node, Express and MongoDB, so that users can save books for review or purchase later.
