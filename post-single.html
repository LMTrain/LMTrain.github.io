Date Checking Program

Posted by Laycon Muriziq Jan 08, 2019 Programming

I have always been interested in writing computer programs.

I wrote my first code in QBasic Language towards the end of 1999. If you can recall, during that period, we are heading to the Year 2000 and there was this Millennium Bug issue about computer dates dating back to 1900. I wrote a program that tackles this issue. I call it "LM DATE CHECKER". This program allows users to check dates 30,000 years (BC) back and 30,000 years (AD) forward. This date checker is good for Historians and Researchers who are interested in a date that matches the day. I still have this program, but unfortunate it will not run on any Windows machine from Windows XP upward. It runs only on Windows 3.1, 95, and 98. I also wrote a game program for brain exercise, which unfortunate has the same faith as the Date Checker, it can't run on modern Windows machine.
